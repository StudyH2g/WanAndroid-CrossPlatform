# 多端移动开发技术对比（Android Compose / Flutter / 微信小程序）
  
本项目基于同一后端 API（WanAndroid），分别使用 Android Jetpack Compose、Flutter 与微信小程序进行实现，掌握了多端开发流程。并通过实践对比，形成了对不同技术栈优劣的直观认识。  
  
## 各端实现特点

**Android Jetpack Compose**：为 Android 原生开发而生的现代声明式 UI 框架，性能与体验最贴近系统，适合对性能和交互有高要求的复杂应用。

**Flutter**：真正的跨平台 UI 框架，使用 Dart 语言和自绘引擎，一套代码可构建多端高性能应用，开发效率高。

**微信小程序**：依托微信生态的轻量级应用，开发门槛低、即用即走，适合快速构建面向微信用户的轻服务与工具。
